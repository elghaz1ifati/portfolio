## 📖 About

This is the initial release of my personal portfolio website. It serves as a showcase of my current skills in React and Tailwind CSS. This version focuses on core functionality and a clean design, with plans to expand features in future updates.

## ✨ Features (MVP)

✅ Responsive Design: Works on mobile, tablet, and desktop.

✅ Project Showcase: Displays my key projects.

✅ Contact Section: Simple form or email link.

✅ Clean UI: Built with utility-first Tailwind classes.

✅ Fast Performance: Optimized React components.

## 🛠 Tech Stack

- **Frontend:** React.js

- **Styling:** Tailwind CSS

- **Build Tool:** Vite React

- **Icons:** React Icons / FontAwesome

## 🚀 Installation

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/yourusername/portfolio-mvp.git

# Navigate to the folder
cd portfolio-mvp

# Install dependencies
npm install

# Start the development server
npm run dev
```
